Classes & Objects

JavaScript Object

Object is a way to group related data and functions (known as methods) together. It's like a container that holds key-value pairs. Each key is a unique name, and its associated value can be any data type, like a string, number, array, or even another function.


            const student = {
                name : 'Jayesh Gavit',
                marks: 85,
                printMarks : function(){
                    console.log(`Name: ${this.name}, Marks: ${this.marks}`);
                }
            };
            console.log(student.printMarks());
OUTPUT : Name: Jayesh Gavit, Marks: 85

Prototype

Object Can inherit properties/methods of another object using __proto__ properties 

            const jayesh = {
                salary : 50000,
            };
            const emp = {
                calTax(){
                    console.log("tax rate of 10% is :", jayesh.salary * 0.10)
                },
            };
    
            jayesh.__proto__ = emp;

            console.log(jayesh.calTax());
OUTPUT : tax rate of 10% is : 5000

Classes in JS

Definition: A class in JavaScript is a blueprint for creating objects. It allows you to define properties and methods in a structured way.

Syntax: Classes are defined using the class keyword.

Constructor: The constructor() method is a special method for creating and initializing an object created with a class.

Methods: Functions inside classes are called methods. You can define methods inside a class, and all instances of the class can use them.

Instantiation: You create new instances of a class using the new keyword. 


            class ToyotaVehi {

                start(){
                    console.log("Car started");
                };
            
                stop(){
                    console.log("Car stopped");
                };
                setBrand(brand){
                    this.brand = brand;
                }
            }
            
            //instance / objects
            let Fortuner = new ToyotaVehi();
            Fortuner.setBrand('Fortuner');
            Fortuner.start();
            Fortuner.stop();
            
            let Hilux = new ToyotaVehi();
            Hilux.setBrand('Hilux');
            Hilux.start();
            Hilux.stop();
OUTPUT : Car started, Car stopped, Car started, Car stopped

Constructor method :

In JavaScript, a constructor function is used to create and initialize objects.

Constructor is a special method used to initialize objects when a new instance of a class or a function is created. The constructor is automatically called when you create an object using the new keyword. Its main purpose is to set up the initial values of an object's properties.

            constructor(brand, mileage){
                this.brand = brand;
                this.mileage = mileage;
            };
