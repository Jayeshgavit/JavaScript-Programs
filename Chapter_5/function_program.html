Function in JavaScript

A function is an independent block of code that performs a specific task, while a function expression is a way to store functions in variables.


Function Declaration and Call:

                function greet(name){
                    document.write(`Hello ${name}`);
                    console.log(" We are Learning JavaScript..! ");
                }
        
                greet("John");

Function Example:

            let a,b;
            function calculator(a, b){
                addi = a + b;
                subt = a - b;
                mul= a * b;
                div = a / b;

                console.log(" Addition of a and b is : ",addi);
                console.log(" Subtraction of a and b is : ",subt);
                console.log(" Multiplication of a and b is : ",mul);
                console.log(" Division of a and b is : ",div);
            }

Arrow Functions : Simple Example

                const sayHello = () => "Hello, World"
                //calling Arrow function
                console.log(sayHello());

Example of one Arg Arrow Function : 

                const square = num => num * num;
                // calling function 
                console.log(square(4));

Examplefor two arguments

                const multiply = (a,b) => a * b;
                console.log("Multiplication of a and b : ",multiply(5,2));

                //or 

                const multiply = (a,b) => {
                    return a * b;
                }
                console.log("Multiplication of a and b : ",multiply(5,2));

Qs. Create a function using the “function” keyword that takes a String as an argument & returns the number of vowels in the string.

                    function countVowels(str) {
                        let count = 0;
                        for (i in str) {
                            if (str[i] === 'a' || str[i] === 'e' || str[i] === 'i' || str[i] === 'o' || str[i] === 'u') {
                                count++;
                            }
                        }
                        return count;
            
                    }
                    console.log("Number of vowels in the string 'Hello World' is : ", countVowels('Hello World'));
