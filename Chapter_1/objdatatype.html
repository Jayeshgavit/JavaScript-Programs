Accessing Object Properties in JavaScript

In JavaScript, object properties can be accessed in two ways:

1. Dot Notation

Dot notation is the most common way to access properties of an object. It is straightforward and easy to read.

Example:

let Student = { name: 'Jayesh', age: 22, grade: 'A' };
console.log(Student.name); // Outputs: 'Jayesh'

Here, we access the name property of the Student object using dot notation: Student.name.

2. Bracket Notation

Bracket notation is used when the property name is dynamic (e.g., stored in a variable) or contains special characters or spaces.

Example:

let Student = { name: 'Jayesh', age: 22, grade: 'A' };
console.log(Student['age']); // Outputs: 22

Here, we use bracket notation to access the age property. In this case, the property name is passed as a string: Student['age'].
